1. Переходим по ссылке https://github.com/KruglovDA/-3278-3246_HW.git
2. Жмем на кнопку Fork (в списке наших репозиториев появился fork)
3. Клонируем к себе в папку
4. Открывает папку в VS
5. выполняем git branch FIO
6. выполняем git checkout FIO
7. Добавляем свой файл с названием FIO.MD
8. Выполняем Коммит
9. выполняем git push (если git ругается, выполняем команду из подсказки)
10. на github выполняем pull request
11. в репозитории того, кому отправили файл, появится новый Pull reguest 
12. Если получателя все устраивает, он делает слияние , нажав merge pull reguest
